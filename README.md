# WanderWords

> #### A blog website to unleash your creativity and share your stories           
>
## Developed with

- Front-End
  - Next.js (App router)
  - Next Navigation
  - React Query
  - Typscript
  - Tailwind css
- Back-End
  - MongoDB
  - node.js
  - express
  - Clerk
  - RESTFUL API
